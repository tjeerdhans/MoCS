# Masters of C# (MoCS)

Masters of C# is a programming tournament framework. It is based on the Masters of Java (MoJ) tournament framework. 

This branch is a work in progress, focusing on the complete rework of the web ui.

## Techniques
  * ASP.NET MVC 5
  * Entity Framework
  * Code Migrations
  * jQuery UI
  * Bootstrap 3

## Prerequisites
  * Latest NUnit
