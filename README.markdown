# Masters of C# (MoCS)

Masters of C# is a programming tournament framework. It is based on the Masters of Java (MoJ) tournament framework. 

It's currently being overhauled and rewritten.

http://tjeerdhans.github.io/MoCS

# OLD info:
## Techniques
  * Entity Framework
  * ASP.NET MVC 2
  * jQuery UI

## Prerequisites
  * NUnit 2.5.2.9222
